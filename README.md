# Bingo service "The Hard Way"
*Greets to Young&&Yandex team!*
## Deployment steps

1. Clone repository

  git clone https://github.com/ruckii/yyapp

2. Install **make** utility

  sudo apt update
  sudo apt install make

3. Do other stuff using **make**

  cd yyapp
  make
